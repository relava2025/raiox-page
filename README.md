# Raio-X Comercial · FACE SCALE

Página pública do diagnóstico comercial para cirurgiões da face.

**No ar em:** https://raiox.gruposystemdigital.com.br

## Como atualizar

1. Abra o editor (`raio-x-editor.html`, no repositório privado `Raio-X`)
2. Edite perguntas, pontuações, lógicas ou identidade visual
3. Clique em **Baixar página pronta**
4. Renomeie o arquivo para `index.html` e substitua aqui (Add file → Upload files)

O GitHub Pages republica automaticamente em ~1 minuto.
